# အမြန် ကိုယ်တိုင်စစ်ဆေးခြင်း (Quick Self-Check) (အမှတ်မပေးပါ) [[quiz2]]

ဘာလဲ?! နောက်ထပ် Quiz လား? ဟုတ်ပါတယ်၊ ကျွန်တော်တို့ သိပါတယ်။ 😅 ဒါပေမယ့် ဒီတိုတောင်းပြီး အမှတ်မပေးတဲ့ Quiz လေးက **သင် ခုနကမှ သင်ယူခဲ့တဲ့ အဓိက အယူအဆတွေကို ပိုမိုခိုင်မာစေဖို့** ကူညီပေးဖို့ ရောက်လာတာပါ။
