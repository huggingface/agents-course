## နိဂုံးချုပ် (Conclusion)

`llama-index` Module ရဲ့ ဒုတိယ Unit ကို အောင်မြင်စွာ ပြီးဆုံးသွားတဲ့အတွက် ဂုဏ်ယူပါတယ် 🥳

သင်ဟာ `llama-index` ရဲ့ အခြေခံသဘောတရားတွေကို ကျွမ်းကျင်သွားပြီဖြစ်သလို၊ ကိုယ်ပိုင် Agentic Workflows တွေကို ဘယ်လိုတည်ဆောက်ရမလဲဆိုတာကိုလည်း သိရှိသွားပါပြီ။

အခု `llama-index` မှာ ကျွမ်းကျင်မှုတွေ ရရှိသွားပြီဆိုတော့၊ သင်စိတ်ဝင်စားတဲ့ Tasks တွေကို ဖြေရှင်းပေးနိုင်မယ့် Search Engines တွေကို စတင်ဖန်တီးနိုင်ပါပြီ။

ဒီ Unit ရဲ့ နောက်ထပ် Module မှာတော့ **LangGraph ကို အသုံးပြုပြီး Agent တွေကို ဘယ်လိုတည်ဆောက်ရမလဲ** ဆိုတာကို လေ့လာသွားပါမယ်။
