# နိဂုံးချုပ် (Conclusion)

ဒီ ဒုတိယမြောက် Unit ရဲ့ `smolagents` Module ကို အောင်မြင်စွာ ပြီးဆုံးသွားတဲ့အတွက် ဂုဏ်ယူပါတယ် 🥳

သင်ဟာ `smolagents` ရဲ့ အခြေခံသဘောတရားတွေကို ကျွမ်းကျင်ပိုင်နိုင်သွားပြီဖြစ်သလို၊ ကိုယ်ပိုင် Agent တစ်ခုကိုလည်း တည်ဆောက်နိုင်ခဲ့ပါပြီ။ အခု `smolagents` မှာ ကျွမ်းကျင်မှုတွေ ရရှိထားပြီဆိုတော့၊ သင်စိတ်ဝင်စားတဲ့ လုပ်ငန်းဆောင်တာတွေကို ဖြေရှင်းပေးမယ့် Agent တွေကို စတင်ဖန်တီးနိုင်ပါပြီ။

> [!NOTE]
> **Agent တည်ဆောက်ခြင်း၏ အရေးပါပုံ:** သင်ဟာ AI နည်းပညာကို အသုံးပြုပြီး ကိုယ်ပိုင် Project တွေ ဖန်တီးချင်တယ်ဆိုရင်၊ ဒီလို Agent Framework တွေကို နားလည်ထားခြင်းက သင့်ရဲ့ စွမ်းဆောင်ရည်ကို အများကြီး မြှင့်တင်ပေးပါလိမ့်မယ်။ `smolagents` လို Framework တွေက LLM တွေရဲ့ စွမ်းအားကို လက်တွေ့ကျတဲ့ လုပ်ဆောင်ချက်တွေ (Actions) အဖြစ် ပြောင်းလဲပေးနိုင်တာကို သင်တွေ့ခဲ့ရပါပြီ။

နောက်လာမယ့် Module မှာတော့ **LlamaIndex ကို အသုံးပြုပြီး Agent တွေကို ဘယ်လို တည်ဆောက်ရမလဲ** ဆိုတာကို ဆက်လက်လေ့လာသွားရမှာ ဖြစ်ပါတယ်။

နောက်ဆုံးအနေနဲ့၊ ကျွန်တော်တို့ရဲ့ Course အပေါ် **သင်ရဲ့ အမြင်တွေကို နားထောင်ချင်ပါတယ်**။ ဒီ Course ကို ဘယ်လို ပိုမိုကောင်းမွန်အောင် လုပ်ဆောင်နိုင်မလဲဆိုတဲ့ တုံ့ပြန်ချက် (Feedback) များ ရှိခဲ့ရင်၊ ကျေးဇူးပြုပြီး 👉 [ဒီ Form ကို ဖြည့်ပေးပါ](https://docs.google.com/forms/d/e/1FAIpQLSe9VaONn0eglax0uTwi29rIn4tM7H2sYmmybmG5jJNlE5v0xA/viewform?usp=dialog)
