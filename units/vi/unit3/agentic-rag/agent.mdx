# Tạo Agent Tổ Chức Gala Của Bạn

Giờ đây khi đã xây dựng xong tất cả các thành phần cần thiết cho Alfred, đã đến lúc kết hợp mọi thứ thành một Agent (tác nhân) hoàn chỉnh có thể giúp tổ chức buổi gala hoành tráng của chúng ta.

Trong phần này, ta sẽ kết hợp các công cụ truy xuất thông tin khách mời, tìm kiếm web, thông tin thời tiết và thống kê Hub thành một Agent (tác nhân) mạnh mẽ duy nhất.

## Lắp Ráp Alfred: Agent Hoàn Chỉnh

Thay vì triển khai lại tất cả các Tools (công cụ) đã tạo từ trước, ta sẽ import chúng từ các module tương ứng đã lưu trong files `tools.py` và `retriever.py`.

<Tip>
Nếu bạn chưa triển khai các Tools (công cụ) này, hãy quay lại các phần <a href="./tools">tools</a> và <a href="./invitees">retriever</a> để thực hiện, sau đó thêm chúng vào files <code>tools.py</code>
