# エージェント入門

<img src="https://huggingface.co/datasets/agents-course/course-images/resolve/main/en/unit1/thumbnail.jpg" alt="Thumbnail"/>

最初のユニットへようこそ。ここでは **AI エージェントの基礎をしっかりと身につけます**。内容は以下の通りです：

- **エージェントの理解**  
  - エージェント（Agent）とは何か、どのように動作するのか？  
  - エージェントはどのように推論と計画を使って意思決定を行うのか？

- **エージェントにおける LLM（大規模言語モデル）の役割**  
  - LLM がエージェントの「頭脳」としてどのように機能するか。  
  - LLM がメッセージシステムを通じてどのように会話を構造化するか。

- **ツールとアクション**  
  - エージェントが外部ツールを使って環境とやり取りする仕組み。  
  - エージェント用のツールを構築・統合する方法。

- **エージェントのワークフロー：** 
  - *思考（Think）* → *行動（Act）* → *観察（Observe）*

これらのトピックを学んだ後、`smolagents` を使って **初めてのエージェントを構築します**！

Alfred という名前のエージェントが簡単なタスクを処理し、これらの概念を実践でどのように適用するかを示します。

さらに、**エージェントを Hugging Face Spaces に公開する方法** も学べるので、友人や同僚と共有できます。

最後に、このユニットの終わりにクイズがあります。合格すると、**最初のコース認定証** である 🎓 エージェント基礎認定証 を取得できます。

<img src="https://huggingface.co/datasets/agents-course/course-images/resolve/main/en/unit1/certificate-example.jpg" alt="Certificate Example"/>

このユニットは **必須の出発点** であり、より高度なトピックに進む前にエージェントを理解するための基盤を築きます。
