

# イントロダクション

<img src="https://huggingface.co/datasets/agents-course/course-images/resolve/main/en/bonus-unit3/pokemon_thumbnail.png" alt="ボーナスユニット3 ゲームにおけるAI"/>

🎶誰よりも強くなりたい...🎶

この**ボーナスユニット**へようこそ。ここでは、**AIエージェント（Agent）とゲーム**のエキサイティングな交差点を探求します！🎮🤖

ノンプレイヤーキャラクター（NPC）がスクリプト通りのセリフをただ読み上げるのではなく、ダイナミックな会話を行い、あなたの戦略に適応し、ストーリーの展開に合わせて進化するゲームを想像してみてください。これこそが**LLMとエージェント的な振る舞いをゲームに組み合わせる**力です。**これまでにない創発的なストーリーテリングとゲームプレイ**への扉を開きます。

このボーナスユニットでは、以下のことを学びます：

- **ポケモン風のターン制バトル**に参加できるAIエージェントの構築方法を学ぶ
- エージェントと対戦したり、オンラインで他のエージェントに挑戦する

AIコミュニティでは、LLMを使ってポケモンをプレイする[いくつかの](https://www.anthropic.com/research/visible-extended-thinking)[例](https://www.twitch.tv/gemini_plays_pokemon)がすでに見られます。このユニットでは、コースを通じて学んだアイデアを使って、自分自身のエージェントでそれを再現する方法を学びます。

<img src="https://huggingface.co/datasets/agents-course/course-images/resolve/main/en/bonus-unit3/claude-plays-pokemon.png" alt="Claudeがポケモンをプレイ"/>

## さらに深く学びたい方へ

- 🎓 **ゲームにおけるLLMをマスターする**：フルコース [Machine Learning for Games Course](https://hf.co/learn/ml-games-course) でゲーム開発をさらに深く学びましょう。

- 📘 **AIプレイブックを手に入れる**：[AI Playbook for Game Developers](https://thomassimonini.substack.com/) で、インサイト、アイデア、実践的なヒントを発見しましょう。インテリジェントなゲームデザインの未来が探求されています。

構築に入る前に、まず**4つのインスピレーションあふれる実例**を通じて、LLMがすでにゲームでどのように活用されているかを見ていきましょう。
