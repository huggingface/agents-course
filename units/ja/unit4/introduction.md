```markdown
# 最終ユニットへようこそ [[introduction]]

<img src="https://huggingface.co/datasets/agents-course/course-images/resolve/main/en/unit4/thumbnail.jpg" alt="AI Agents Course thumbnail" width="100%"/>

コースの最終ユニットへようこそ！🎉

ここまでで、AI Agent の構成要素の理解から独自の Agent の作成まで、**AI Agent に関する確かな基礎を築いてきました**。この知識をもとに、**強力な Agent を構築し**、急速に進化するこの分野の最新動向を把握する準備が整いました。

このユニットは、学んだことを実践に活かすことがすべてです。これは**最後のハンズオンプロジェクト**であり、これを完了することが**コース修了証**を取得するための条件です。

## チャレンジの内容は？

独自の Agent を作成し、**[GAIA ベンチマーク](https://huggingface.co/spaces/gaia-benchmark/leaderboard)のサブセットを使用してそのパフォーマンスを評価**します。

コースを正常に修了するには、Agent がベンチマークで**30%以上のスコア**を達成する必要があります。それを達成すれば、あなたの専門知識を公式に認定する**修了証**を獲得できます。🏅

さらに、他の受講者と比較してみましょう！専用の**[受講者リーダーボード](https://huggingface.co/spaces/agents-course/Students_leaderboard)**が用意されており、スコアを提出してコミュニティ全体の進捗を確認できます。

> ** 🚨 注意：上級者向け＆ハンズオンユニット**
>
> このユニットは、より実践的でハンズオンなアプローチに移行します。このセクションで成功するには、**より高度なコーディング知識**が必要であり、コースの前半と比べて**明示的なガイダンスが少ない**中でタスクを進めることが求められます。

ワクワクしませんか？さあ、始めましょう！🚀
```
