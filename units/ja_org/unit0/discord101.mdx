## Hugging Face Discord コミュニティでの Agents コース

Discord を始めたばかりだと少し圧倒されるかもしれませんので、ナビゲーションに役立つ簡単なガイドをご紹介します。

HF コミュニティサーバーには、さまざまな分野に関心を持つ活発なコミュニティがあり、論文ディスカッションやイベントなどを通じて学ぶ機会が提供されています。

[サインアップ](http://hf.co/join/discord)後、`#introduce-yourself` チャンネルで自己紹介をしましょう。

Agents コース用に4つのチャンネルを作成しました：

- `agents-course-announcements`：**最新のコース情報**をお届けします。
- `🎓-agents-course-general`：**一般的なディスカッションや雑談**の場です。
- `agents-course-questions`：**質問をしたり、クラスメイトを助けたり**するためのチャンネルです。
- `agents-course-showcase`：**あなたの最高のエージェントを披露**する場です。

さらに以下もチェックできます：

- `smolagents`：**ライブラリに関するディスカッションやサポート**のチャンネルです。

## Discord を効果的に使うためのヒント

### サーバーへの参加方法

Discord にあまり馴染みがない場合は、サーバーへの参加方法について<a href="https://support.discord.com/hc/en-us/articles/360034842871-How-do-I-join-a-Server#h_01FSJF9GT2QJMS2PRAW36WNBS8" target="_blank">こちらのガイド</a>を参照してください。

手順の簡単なまとめ：

1. <a href="https://discord.gg/UrrTSsSyjb" target="_blank">招待リンク</a>をクリックします。
2. Discord アカウントでサインインするか、アカウントがない場合は新規作成します。
3. AI エージェントではないことを確認します！
4. ニックネームとアバターを設定します。
5. 「サーバーに参加」をクリックします。

### Discord を効果的に使う方法

Discord を効果的に使うためのヒントをいくつかご紹介します：

- **ボイスチャンネル**も利用可能ですが、テキストチャットがより一般的に使われています。
- **Markdown 形式**でテキストをフォーマットできます。コードを書く際に特に便利です。ただし、リンクに関しては Markdown がうまく機能しない場合があります。
- **長い会話**にはスレッドを活用して、ディスカッションを整理しましょう。
