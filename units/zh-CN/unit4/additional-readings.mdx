# 接下来学什么？我应该学习哪些主题？

智能体 AI 正在快速发展，理解基础协议对于构建智能自主系统至关重要。你需要熟悉的两个重要标准是：

- **模型上下文协议（Model Context Protocol，MCP）**  
- **智能体间协议（Agent-to-Agent Protocol，A2A）**

## 🔌 模型上下文协议（MCP）

Anthropic 提出的**模型上下文协议（Model Context Protocol，MCP）**是一个开放标准，它使 AI 模型能够安全无缝地**连接外部工具、数据源和应用程序**，从而增强智能体的能力和自主性。

可以将 MCP 想象成一个**通用适配器**，就像 USB-C 接口一样，它允许 AI 模型接入各种数字环境**而无需为每个环境定制集成方案**。

MCP 正在快速获得行业认可，OpenAI 和 Google 等主要公司已开始采用该协议。

## 🤝 智能体间协议（A2A）

Google 开发的**智能体间协议（Agent-to-Agent Protocol，A2A）**是 Anthropic MCP 的互补协议。MCP 专注于连接智能体与外部工具，而 **A2A 则实现智能体之间的互联**，为构建协作式多智能体系统铺平道路，这些系统能够协同解决复杂问题。
